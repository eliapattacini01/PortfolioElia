E-Commerce Chatbot (LLM)

A conversational AI assistant for e-commerce FAQs, powered by a large language model. It answers product questions, assists with order tracking, and guides users through the shopping experience, deployed as a live Streamlit app.

Open Live App

Goal

Build a practical LLM-powered chatbot that handles e-commerce use cases: product discovery, FAQs, and conversational guidance.

LLMConversational AIE-Commerce

LLM & RAG Design

Retrieval-Augmented Generation (RAG) grounds the model's responses in a product catalog, minimising hallucinations and keeping answers accurate and relevant without expensive fine-tuning.

RAGEmbeddingsVector Search

Deployment

Shipped as an interactive Streamlit app with a clean chat UI, session memory, and instant public access — ready to demo and iterate on without infrastructure overhead.

What I Built

Chatbot Pipeline

  • User query → retrieval from product knowledge base
  • Context injection into LLM prompt
  • Response generation with conversation history
  • Streamlit session state for multi-turn memory

Knowledge Base

  • Product catalog chunked and embedded
  • Similarity search to find relevant context
  • Prompt engineering to reduce hallucinations
  • Fallback handling for out-of-scope queries

Evaluation

  • Qualitative testing across query types
  • Relevance and faithfulness checks
  • Iterative prompt tuning based on failure cases
  • Latency profiling for user experience
